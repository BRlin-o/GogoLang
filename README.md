# GogoLang

## Installation

```bash
python -m venv .venv
```

- Windows
    ```bash
    cd .venv/Scripts
    activate.bat

    cd ../../
    ```

- MacOS
    ```bash
    source .venv/bin/activate
    ```

### Install python package

- Core
    ```bash
    git clone https://github.com/BRlin-o/GogoLang-core core
    pip install -r core/requirements.txt
    ```

```bash
pip install -r requirements.txt
```

## Run Scription

streamlit run web_app.py
