# GogoLang

## Installation

```bash
python -m venv .venv
```

- Windows
```bash
python -m venv .venv
cd .venv/Scripts
activate.bat

cd ../../
```

- MacOS
```bash
source .venv/bin/activate
```

### Install python package

```bash
source .venv/bin/activate
pip install -r requirements.txt
```

## Run Scription

```bash
python web_app.py
```
